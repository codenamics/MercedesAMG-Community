# MercedesAMG-Community

NodeJS backend for AMG community app

## Main Technologies

* [x] **JavaScript ES5/ES6+**
* [x] **NodeJS**
* [x] **Express**
* [x] **JWT**
* [x] **Passport**
* [x] **MongoDB**


### Setup

```bash
$ git clone or download
```

Go to project direction

Install dependencies

```bash
$ npm install

```
Run server (nodemon)

```bash
$ npm run start

```
